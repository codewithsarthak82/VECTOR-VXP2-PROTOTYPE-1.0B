# Project Brief: VECTOR VXP2

1. **Project:** VECTOR VXP2 (Orion Spacetech).

2. **Objective:** Predicting RUL for rocket engines using a Hybrid LSTM + PINN approach.

3. **Data Source:** NASA CMAPSS (FD001/FD002/FD004).

4. **Physics Constraints:** 1-D Isentropic Baseline, Fanno Flow (friction), and Rayleigh Flow (heat).

5. **Current Goal:** Transition from Random Forest (1.0) to LSTM-based time-series prediction (2.0).
